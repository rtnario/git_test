File Structure Tips

  • Directory Names
    • img/images: images directory
    • css: stylesheets directory
    • js/javascript: scripts directory
  • the shorter the link, the easier to access
  • less weird names for less probability to have malformed URL
  • avoid spaces, capitals, numbers
